Create `.env` file in the root directory and add the following environment variables:

```bash
TEST_TELEGRAM_TOKEN=your_bot_token
TEST_USER_ID=your_user_id
TEST_GROUP_ID=your_group_id
```

```bash
npm test
npm run lint
npm run format:check
```
